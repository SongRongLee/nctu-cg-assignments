# nctu-cg-assignments
Programming assignments for NCTU Computer Graphics Class. Course No.1184.

**Course No:** 1184  
**Semester:** 1061  
**Teacher:** 莊榮宏  

## General Key Usage
* **'w'**: zoom in  
* **'a'**: move left (circle the center)  
* **'s'**: zoom out  
* **'d'**: move right (circle the center)  

* **'1'~'9'**: select n-th object  
* **Mouse**: drag the selected object.  
* **Right**: increase x  
* **Left**: decrease x  
* **Up**: increase y  
* **Down**: decrease y
